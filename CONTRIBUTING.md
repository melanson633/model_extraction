# Contributing to AM Model Extraction

Thank you for considering contributing to this project. The following guidelines will help ensure a smooth development process.

## How to Contribute

1. **Fork the repository** and create your feature branch:
   ```bash
   git checkout -b feature/my-feature
   ```
2. **Make your changes** following the code style used in the repository.
3. **Add tests** for new features whenever possible.
4. **Run the test suite** before submitting your changes:
   ```bash
   pytest
   ```
5. **Commit your changes** with clear messages and push to your fork:
   ```bash
   git commit -m "Add my feature"
   git push origin feature/my-feature
   ```
6. **Open a Pull Request** on GitHub and describe your changes.

## Code Style

- Use `black` for formatting and `flake8` for linting.
- Include docstrings for all public functions.

## Reporting Issues

If you encounter a bug or have a feature request, please open an issue on GitHub with details and steps to reproduce the problem.
